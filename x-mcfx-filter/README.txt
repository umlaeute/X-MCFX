This Plug-in is a multichannel EQ with Low Cut, High Cut and 2 Peak Filters

High and Low Cut are 2nd order Butterworth Filters and can be selected as one stage (2nd order) or 2x2nd order cascaded.
Resulting in a Linkwitz Riley Filter suitable for using it as phase aligned X-Over Filters.
